Site dedicado à instalação da Tourmed

Instruções de Instalação

1 - Baixe o arquivo zipado.

2 - Coloque-o na raiz do seu computador. A raiz do computador é o disco local, OS (C:)

3 - Descompacte o arquivo para a pasta TourMed.

4 - Abra TourMed+.exe para utilizar o sistema.


É importante seguir esse passo a passo para garantir que o software seja atualizado corretamente no futuro. Coloque o arquivo na raiz do seu computador para evitar perda de informações nas futuras atualizações.

Qualquer dúvida, estarei a disposição.
